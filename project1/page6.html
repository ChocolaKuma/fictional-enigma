Love Plus


So lets begin. Love Plus is a pretty much generic JRPG originally
written and release in Japan. The object of the game is to more or
less , raise your stats to the point where you can trigger an
event where you talk to one of three girls. I chose the route of Manaka.
the tennis club misfit. Over all I enjoyed it despite having at least 30 problems getting it to run at any given time.
Then it would only save once every two
hours so any way. This game was very enjoyable. I've always enjoyed
dating sims and this is another one to add to the list.

~Update~
~(1-25-2014)~

Graphics 7/10(I was a little glitchy on my devise)
Concept 2/10 (Nothing new in this one)
Translation 9/10 (I prefer the Horifics like kun,san,sama, ect.)
Replay value 10/10 (I ended up completeing all three story lines)
